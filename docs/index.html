よくあるご質問 faq

q 日本に勤務してもらうことは可能ですか？
a 弊社では日本にエンジニアを派遣するサービスを行っていないため対応致しかねます。
q 契約前にエンジニアと面談は可能ですか？
a 可能です。
弊社に既に在籍しているエンジニア、タレント検索に掲載されているエンジニア両者とも事前に面談が可能です。
q コミュニケーションツールは何を使用しますか？
a 貴社に合わせて対応可能です。
特に指定がなければ、Slack を利用しています。
それ以外の管理ツールも標準のものがありますが、柔軟に対応しますので、ご相談ください。
q このサービスの利点はなんですか？
a 私たちの強みは、自社サービスを行う際へのサポート体制です。
早いペースで拡大したい、コストは抑えたい、採用に人員を割けない、スポットでCTO業務を任せたい、など今後の拡大と長期間の運用に見据えたお客様にマッチすると思っています。役員である 3名とも自社サービスでの起業経験があり、経営目線でのサービス立ち上げの支援も強みです。
q なぜフィリピンを選ぶべきですか？
a 日本とフィリピンを比べた場合、その他他国とフィリピンを比べた場合を説明します。
日本とフィリピンでの大きな違いは、採用のしやすさ、コストです。現在の環境では、素早くチームを拡大できることは大きなアドバンテージになり、大きなケースでは、約半分のコストで、同クオリティの開発が可能だと実感しています。
また、他国で特によくオフシュアで使われているベトナムと比べますと、コスト面ではあまり変わらないかなと思います。大きく異なるのは英語です。彼らは、英語を話すことができません。ここで通訳を介さずにコミュニケーション取るときや御社の中で外国人エンジニアを入れてグローバルチームを作る時なども英語が話せるフィリピンは強みになります。
q 外国人エンジニアの良い点、難しい点を教えてください。
a 外国人エンジニアの良い点は、なんと言っても豊富なタレントから採用できることです。
日本では、エンジニアの給与の高騰が激しく、そこまで技術力がないエンジニアでもかなりの給与をもらっています。そのような状況と比べると、費用対効果はいいと思います。素早く採用できるのも魅力です。難しい点は、コミュニケーションと質ややり方への文化の違いです。ここは、全てがマイナス要素ではないですが、やりずらさを感じます。ここはしっかりと文化を理解し、良いやり方を見つけることで問題を最小化できると考えていますが、同じやり方ではいけないという部分はありますね。
q 御社独自の強みはありますか？
a 私たちの強みは、自社サービスを行う際へのサポート体制です。
早いペースで拡大したい、コストは抑えたい、採用に人員を避けない、スポットでCTO業務を任せたい、など今後の拡大と長期間の運用に見据えたお客様にマッチすると思っています。役員である 3名とも自社サービスでの起業経験があり、経営目線でのサービス立ち上げの支援も強みです。
q どんなプログラミング言語に対応できますか？
a 特に、強いのは、PHP, Nodejs です。
フロントエンドでは、React, Vue をよく用いています。その他にもRuby, Python, Typescript, ReactNative, Flutter を使ったアプリ開発なども得意な分野です。
q どのように現地メンバーとコミュニケーションできますか？
a ブリッジエンジニアを活用して、日本語でのコミュニケーションや、英語を使い直接コミュケーションをすることが可能です。
また、ツールとしては、チャットツールをメインに使い、定期的、適宜、ビデオ会議を入れる場合が多いです。お客様の必要に応じ、現地で一緒に仕事することもできます。
q ブリッジエンジニアとはなんですか？
a ブリッジエンジニアは、開発チームを運用するノウハウを持った日本人または日本語が話せる日本側とフィリピンエンジニアの架け橋となるエンジニアです。ディレクターと呼ばれるポジションに近いです。
q 人材は面接できますか？
a はい。面談可能です。
q もしエンジニアが良くなかったらどうなりますか？
a メンバーの変更が可能です。
q 料金について詳しく説明してください。
a 料金は、とてもシンプルで、人員料金表の月額料金と、その方々が使う備品代のみになっています。
その他の費用は一切かかりません。
q いつからスタートできますか？
a 弊社のプールメンバーからの採用になれば、最速 2週間後、外からとなれば、1-1.5ヶ月いただいています。
q まけてもらえませんか？
a 申し訳ありません。お値引きはできないことになっています。
q 契約期間短くできませんか？
a 申し訳ありません。契約期間は最低半年となっております。
ただし、人員が合わない、期待通りでなければ、変更は可能です。
